Home Assistant integration for Yuno Energy Heat.

## Installation

### HACS

1. Go to HACS -> Integrations -> 3 dots (top right) -> Custom repositories.
2. Add `https://github.com/Irishsmurf/ha-yunoheat` with category `Integration`.
3. Click "Download" on the Yuno Energy Heat card.
4. Restart Home Assistant.

### Manual

1. Copy the `custom_components/yunoheat` directory to your `custom_components` directory.
2. Restart Home Assistant.

## Configuration

Configuration is done via the Home Assistant UI. Go to Settings -> Devices & Services -> Add Integration and search for "Yuno Energy Heat".
